# 自己紹介サイト

HTMLとCSSを使用して制作した、1ページ構成の自己紹介サイトです。

## 概要

プロフィール、スキル、制作物、問い合わせ情報を掲載しています。

レスポンシブデザインに対応しており、PC・スマートフォンの両方で見やすいレイアウトを意識しました。

## 使用技術

- HTML5
- CSS3
- Flexbox
- Media Query
- Git / GitHub

## 主な機能

- セマンティックタグを使用したページ構成
- 外部CSSによるデザイン適用
- Flexboxを利用したレイアウト
- 画像表示とalt属性の設定
- スムーズスクロール
- レスポンシブ対応
